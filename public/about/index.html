About

This is a “hello world” example website for the blogdown package. The theme was forked from @jrutheiser/hugo-lithium-theme and modified by Yihui Xie.

This website can be found here, and the corresponding github repository here. I tried to set up an easy process to get a hosted github page set up. Here what I learned so far. First, in connection with RStudio and github, it’s rather easy.

Lets go through the workflow:

  1. Set up a blogdown project infrastructure, which is described here
  2. Set up a new github repository (no readme), and link the git to the Rtudio. This involves some Shell, and is nicely described here

  • Netlify: Then it pretty much up where you want to host it. You can put it on netlify, which is easy when everything is on github and you have an account there.

  • GitHubPages: Another alternative is to host it via github pages. This seems easy and appropriate for your personal website or smll course websites without too much infrastructure.

    • The general infrastructure is decribed here
    • Generally, GitHubPages is only supposed to work with Jekill to build sites, but it turns out most Hugo tenplates work just fine. A nice description how to get started can be found here
    • In case you just want to render a website ()such as a revealjs presentation, its even easier as described here

    Have fun!
